# Skill Tricks Library

Tricks are micro-learnings discovered during real work. Each trick belongs to a skill
domain and accumulates over time. During skill selection, read the relevant domain's
tricks before executing.

**Format**: `- **Name** (date | context): description`

---

## _general
- **Read before write** (2025-01-25 | all tasks): Never edit a file you haven't read in this session. Even if you "know" what's there — files change between sessions.
- **Absolute paths only** (2025-01-25 | file operations): Always use absolute paths in tool calls. Relative paths break when working directory shifts.
